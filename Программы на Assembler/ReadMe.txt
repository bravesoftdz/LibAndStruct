На компьютере:

Для запуска нужно создать и настроить проект в visual studio.
В visual studio 2013 так:
Файл -> Создать проект -> Visual C++ -> Win32 -> Консольное приложение Win32
Обозреватель решений -> ПКМ <Имя проекта> -> Зависимости сборки -> Настройки сборки -> ФЛАГ masm(.targets, .props)
ПКМ Файлы исходного кода -> Добавить -> Создать элемент -> <имя.расширение>
ПКМ <создынный файл> -> Свойства -> Тип элемента -> Microsoft Macro Assembler.

Если будкт ошибка: "error LNK1281: Не удается создать образ SAFESEH.", просто отключи SAFESEH в настройках.

; Минимально-возможная программа на masm
.686p
.model flat

.code
_main:
ret 0
end _main
	
; Hellow world программа на masm
.686p
.model flat
includelib msvcrt.lib
extern __imp__printf: proc

.data
fmt db "Hellow world", 10, 0

.code
_main:
push offset fmt
call dword ptr __imp__printf
add esp, 4
ret 0
end _main

; Сумма двух чисел с выводом в консоль на masm
.686p
.model flat
includelib msvcrt.lib
extern __imp__printf: proc

.data
fmt db "Sum = %d", 10, 0
n dword 0
m dword 0

.code
_main:
mov n, 10
mov m, 3
mov eax, n
add eax, m
mov n, eax
push n
push offset fmt
call dword ptr __imp__printf
add esp,8
ret 0
end _main

Требования к лексическому анализатору:
	1. работа с командной строкой;
	2. запуск с ключами (о программе, о версии, лексический анализатор);
	3. написать тесты;
	4. выводить сообщения в стандартный поток ввода-вывода.

Типы токкенов
	1) идентификаторы
	1.1) ключевые слова
	2) числа
	2.1) целые
	2.2) вещественные
	3) строки / символы
	4) операции
	5) разделители
	6) пробельные символы
	6.1) комментарии

В режиме лексического анализатора, пользователю должна выдаваться таблица с 4-я столбцами:
|тип лексемы|позиция лексемы|текст лексемы|значение лексемы|
